# misc_files

This directory contains a collection of various notes I took while a graduate student in particle physics at Duke University.  I make no claims regarding their accuracy, quality, or completeness.

## Catalog

_(in alphabetical order)_

- [box_diagram.pdf](box_diagram.pdf)
  - Incomplete calculation of a box diagram for two heavy quarks using the threshold expansion. I don't remember why I did this; it might have been related to [nrqcd_threshold_expansion.pdf](nrqcd_threshold_expansion.pdf).
- [defense.pdf](defense.pdf)
  - Some notes I made while brushing up on some topics before my dissertation defense.
- [defense_slides_draft.pdf](defense_slides_draft.pdf)
  - A draft of my slides for my dissertation defense. 
- [fragmentation_functions_gss.pdf](fragmentation_functions_gss.pdf)
  - Notes I made in preparation for a Graduate Student Seminar at Duke University. I was attempting to give a very basic understanding of transverse-momentum-dependent fragmentation functions, starting from the quantum harmonic oscillator and proceeding through some very hand-wavey derivations.
- [nrqcd_threshold_expansion.pdf](nrqcd_threshold_expansion.pdf)
  - Some preliminary notes I made for a potential research project on NRQCD matching using the threshold expansion. This work never really panned out.
- [schwartz.pdf](schwartz.pdf)
  - Notes I took on ["Quantum Field Theory and the Standard Model"](https://doi.org/10.1017/9781139540940) by Matthew D. Schwartz.
- [tmd_handbook.pdf](tmd_handbook.pdf)
  - Notes I took on the first couple chapters of the [TMD Handbook](https://inspirehep.net/literature/2650019).
